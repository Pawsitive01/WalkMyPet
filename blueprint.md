# Walk My Pet - Application Blueprint

## Overview

Walk My Pet is a mobile application that connects dog walkers with pet owners. The app provides a platform for owners to find trustworthy and experienced walkers, and for walkers to offer their services to a wider audience. The application features a modern, intuitive user interface with a focus on user experience and a beautiful, data-driven design.

## Style and Design

The application follows a modern and clean design aesthetic, with a focus on usability and visual appeal. The following design elements have been implemented:

*   **Color Palette:** A vibrant and energetic color palette is used to create a visually engaging experience. The primary color is a modern indigo, which is used for branding and key UI elements. The color scheme is designed to be accessible and visually appealing in both light and dark modes.
*   **Typography:** The app uses a combination of the Poppins and Inter fonts from Google Fonts to create a clean and readable typography. The font sizes are used to create a clear visual hierarchy and to emphasize important information.
*   **Iconography:** The app uses modern, interactive iconography to enhance the user's understanding and logical navigation of the app. The icons are from the Material Design icon set and are used to represent key actions and information.
*   **Layout:** The app uses a visually balanced layout with clean spacing to create a user-friendly and intuitive experience. The layout is designed to be mobile responsive and to adapt to different screen sizes.
*   **Animation:** The app uses subtle animations and transitions to create a more engaging and interactive user experience. The animations are designed to be smooth and to provide visual feedback to the user.

## Features

The application includes the following features:

*   **Walker and Owner Lists:** The app displays separate lists of dog walkers and pet owners, allowing users to easily browse and find the right match for their needs.
*   **Profile Details:** Users can view detailed profiles of walkers and owners, including their name, photo, rating, reviews, and a short bio.
*   **Hero Animations:** The app uses Hero animations to create a seamless transition between the list view and the detail view, providing a more engaging and visually appealing user experience.
*   **Dark Mode:** The app includes a dark mode option, which can be toggled by the user. The dark mode is designed to be easy on the eyes and to provide a more comfortable viewing experience in low-light environments.
*   **Modern UI:** The app features a modern user interface with a custom app bar, a bottom navigation bar, and custom card widgets for displaying walker and owner information.
